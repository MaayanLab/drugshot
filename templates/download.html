Download Section

The download section contains all data that was compiled for DrugShot. It contains various drug pair similarity matrices as well as a preprocessed DrugRIF file. Drug association to biological terms is performed using the NCBI e-utilities API for PubMed.

DrugRIF

DrugRIF was built by querying PubChem with all small molecules from Drugmonizome and SEP-L1000 using the PUG-REST API. All PubMed IDs matching the small molecule query with associated publications are contained in this file. The file contains 2,121,206 small molecule associations to PubMed IDs.

Pairwise Drug L1000 Signature Similarity Matrix

The pairwise drug L1000 signature similarity matrix was calculated using the L1000 drug-induced signature data from SEP-L1000. Differential gene expression was calculated using the Characteristic Direction method, and pairwise cosine similarity was computed between all drug signatures.
